# 模型路由（可替換操作知識）

> 本檔不是流程鐵律。每季或供應商重大發布後，依官方文件更新可用模型、價格、deprecation 與工具支援；不可使用會熱切換的 `latest` 作生產自動化依賴。

| 任務 | 預設能力等級 | 升級條件 |
|---|---|---|
| 純文字、格式、狀態同步 | 經濟型／deterministic automation | 語意會改變規則時升級 |
| 一般規劃、實作、review | 主力型 | 跨模組、未知根因或高風險 |
| 安全、金流、資料正確性 | 高階型 + 跨家族 review | 人工 sign-off 不可省略 |
| 部署、migration 異常 | 主力型 | 不可逆風險或根因不明 |

先選風險，再選供應商；高能力模型不取代測試、平台閘門或獨立審核。各專案可在自己的 `MODEL_ROUTING.md` 記載可用 model ID、成本與 fallback。
